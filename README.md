# html-css-js-portfolio
🚀 George Pavlos - Personal Portfolio
This repository hosts my personal portfolio website, built to showcase my skills, experience, and projects as a Rising Software Developer.

The site is designed to be fully responsive, providing a clean and professional view of my journey and capabilities.

✨ Features
Responsive Navigation: Includes a standard desktop navigation bar and a hidden hamburger menu for screen sizes under 1200px (defined in mediaqueries.css).

Smooth Scrolling: Implements smooth transitions between sections (About, Experience, Projects, Contact).

Detailed Experience Section: Clearly outlines technical skills (HTML, CSS, JavaScript, Python, React, Git/GitHub, Node.js) with defined proficiency levels.

Project Demos: Each project card provides direct links to the GitHub repository and a Live Demo link.

Contact Information: Easy access to my email and LinkedIn profile.

🛠️ Technologies Used
This project is built using foundational web development technologies:
HTML5: Structure and semantic organization.
CSS3: Styling and layout, including custom utility classes and responsiveness.
JavaScript (Vanilla): Used primarily for the mobile hamburger menu functionality (script.js).
